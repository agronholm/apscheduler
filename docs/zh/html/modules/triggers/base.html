apscheduler.triggers.base¶

API¶

class apscheduler.triggers.base.BaseTrigger¶

Abstract base class that defines the interface that every trigger must implement.

get_next_fire_time(previous_fire_time, now)¶

Returns the next datetime to fire on, If no such datetime can be calculated, returns None.

Parameters:
  • previous_fire_time (datetime.datetime) – the previous time the trigger was fired
  • now (datetime.datetime) – current datetime
